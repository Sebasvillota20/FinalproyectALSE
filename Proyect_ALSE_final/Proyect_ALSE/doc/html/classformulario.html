Detailed Description

En esta clase se implementan los métodos usados en la ventana llamada formulario, donde se muestra información del usuario como su edad y el usuario con el que se registró, además, se registran los pacientes de los que se obtiene la información socio económica. luego, se llama la ventana main_app en la que se puede realizar la prueba de agilidad.
